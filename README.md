
# Burger Restaurant's Webpage 

A website for a hamburger restaurant where customers may place orders for delectable burgers and beverages. 



## Features

- Login and Sigup page 
- Menu page 
- Reservation page


## Tech Stack

**Client:** HTML, CSS
